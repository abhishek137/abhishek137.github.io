How to make docker work in Windows

2016-08-06

I tried the new Docket Toolbox for windows. However, the Docker Start Terminal never seemed to work for me. I keep on getting below error, nothing seems to resolve it.

Looks like something went wrong in step 'Checking if machine default exists'...
Press any key to continue...

From the message it looked it something went wrong with the virtual machine, so I decide to invesgigate myself.

I executed below command in a cmd to list down the virtual machines available.

> docker-machine ls

NAME         ACTIVE   DRIVER       STATE     URL   SWARM   DOCKER    ERRORS
my-default   -        virtualbox   Stopped                 Unknown

For some reason, the default VM had not started. I started it manually by executing

> docker-machine start my-default
Starting "my-default"...
(my-default) Check network to re-create if needed...
(my-default) Waiting for an IP...
Machine "my-default" was started.
Waiting for SSH to be available...
Detecting the provisioner...
Started machines may have new IP addresses. You may need to re-run the `docker-machine env` command.

Waiting for an IP... took a while

Running docker ps or any docker command gave below error

An error occurred trying to connect: Get http://%2F%2F.%2Fpipe%2Fdocker_engine/v
1.23/containers/json: open //./pipe/docker_engine: The system cannot find the fi
le specified.

To fix this, I had to set appropriate environment variables by executing

> docker-machine env my-default

SET DOCKER_TLS_VERIFY=1
SET DOCKER_HOST=tcp://192.168.99.100:2376
SET DOCKER_CERT_PATH=C:\Users\abhishek_majhi\.docker\machine\machines\my-default

SET DOCKER_MACHINE_NAME=my-default
REM Run this command to configure your shell:
REM     @FOR /f "tokens=*" %i IN ('docker-machine env my-default') DO @%i

Then execute the command suggested in the output.

> @FOR /f "tokens=*" %i IN ('docker-machine env my-default') DO @%i

Now run docker ps or any docker command. It should work now.

> docker ps
CONTAINER ID        IMAGE               COMMAND             CREATED             STATUS              PORTS               NAMES
