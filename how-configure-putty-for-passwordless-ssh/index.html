How configure Putty for passwordless SSH

2016-08-06

Maintaining multiple servers at work has been a real pain in the ass for me these days. Although I have listed all those servers in putty, I have to remember username-password pairs for each server, or atleast copy it from my secret secure notes when I have to login.

Then I came up with a grand plan for using RSA keys for these logins instead. This made the login process a click of a button in putty.

Generate a key pair with Putty Key Generator

  • Open up puttygen to generate a keypair
  • Generate a SSH-2 RSA keypair
  • Save public key as .pub and private key as .ppk

Add public key as an authorized key in the server

  • Login onto your server using putty
  • Navigate to .ssh folder of current user 
    cd ~/.ssh
