How to point your sub-domain to a Vultr instance

2017-06-26
cloud, dns, vultr

I recently hosted one of my nodejs web application on Vultr as it provides almost double power for the same price as compared to Digital Ocean. However, configuring sub-domains on Vultr DNS is not supported, unlike Digital Ocean.

I wanted my new web application to be accessible on one of my sub-domain like sub.mydomain.com. To achieve this, I created an A record on DNS of mydomain.com pointing to the public IP of the Vultr instance.

sub          IN  A     192.0.2.3             ; IPv4 address of Vultr instance for sub.mydoamin.com
